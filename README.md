# OSINT Telegram Map for 45thbrigade

This project is a simple, GitHub Pages-friendly map for displaying Telegram-related OSINT observations for 45thbrigade.

## What it does
- Shows a Leaflet map with location-based markers.
- Filters entries by category.
- Uses a JSON data file so you can swap in your own records easily.

## How to publish on GitHub Pages
1. Upload the repository to GitHub.
2. Open the repository settings.
3. Go to Pages.
4. Select the main branch as the source.
5. GitHub Pages will publish the site at a URL similar to `https://your-username.github.io/your-repo/`.

## Customize the data
Edit [data/entries.json](data/entries.json) and replace the sample entries with your own Telegram-monitoring observations.
